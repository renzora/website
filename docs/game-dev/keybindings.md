# Keyboard Shortcuts

Quick reference for all editor keyboard shortcuts.

## Camera

| Key | Action |
|-----|--------|
| `W/A/S/D/E/Q` | Fly camera (hold right-click) |
| `Shift` | Move faster |
| `F` | Focus selected |
| `Numpad 1/3/7` | Front / Right / Top view |
| `Ctrl+Numpad` | Opposite views |
| `Numpad 5` | Toggle perspective/ortho |

## Tools

| Key | Action |
|-----|--------|
| `Q` | Select mode |
| `W` | Translate (move) |
| `E` | Rotate |
| `R` | Scale |
| `G` | Grab (Blender-style) |

## Selection

| Key | Action |
|-----|--------|
| Left Click | Select |
| `Ctrl+Click` | Toggle selection |
| `Shift+Click` | Add to selection |
| `Escape` | Deselect all |
| `Ctrl+D` | Duplicate |
| `Delete` | Delete |

## File

| Key | Action |
|-----|--------|
| `Ctrl+N` | New scene |
| `Ctrl+O` | Open scene |
| `Ctrl+S` | Save scene |
| `Ctrl+Shift+S` | Save as |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## View

| Key | Action |
|-----|--------|
| `Z` | Wireframe |
| `Shift+Z` | Toggle lighting |
| `H` | Toggle grid |

## Play

| Key | Action |
|-----|--------|
| `F5` | Play / Stop |
| `Shift+F5` | Run scripts only |
